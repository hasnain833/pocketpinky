# Pocket Pinky (PinkyPocket)

Your AI Big Sister for Dating Clarity — vet men, decode texts, and date with standards.

## Tech stack

- **Vite** — build tool
- **React** + **TypeScript**
- **Tailwind CSS** + **shadcn/ui**
- **React Router** — routing
- **Framer Motion** — animations

## Getting started

```sh
# Install dependencies
npm i

# Start dev server
npm run dev
```

## Scripts

| Command        | Description              |
|----------------|--------------------------|
| `npm run dev`  | Start dev server         |
| `npm run build`| Production build         |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint               |
| `npm run test` | Run Vitest               |

## Project structure

- `src/pages/` — page components (Index, NotFound)
- `src/components/` — reusable sections and UI (Navbar, Hero, Quiz, etc.)
- `src/components/ui/` — shadcn UI primitives
- `src/lib/` — utilities
